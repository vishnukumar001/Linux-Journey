# Linux Basics

## What is Linux?

Linux is a free and open-source operating system based on Unix.

## Why Learn Linux?

- Used by cybersecurity professionals
- Used by hackers and penetration testers
- Runs most servers worldwide
- Used in cloud computing
- Required for DevOps

## Popular Linux Distributions

- Ubuntu
- Kali Linux
- Debian
- Fedora
- Arch Linux
